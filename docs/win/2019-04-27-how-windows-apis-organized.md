# How Windows APIs organized

## story

一名开发者，不论是进行应用层面的开发，还是内核层面的开发。频繁或偶尔的都需要使用到系统环境提供的一些功能，最常见的如发起一次网络连接、处理硬件传来的中断请求等。而这些本质上是向作为软硬件资源管理角色的操作系统请求资源的调用，是脱离于语言本身存在的。是通过一组接口向操作系统发起和接收，这组接口就是操作系统 API。
